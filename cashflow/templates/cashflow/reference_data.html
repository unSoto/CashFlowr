Управление справочниками

Как работать со справочниками
Порядок создания:
  1. Сначала создайте Типы (Пополнение, Списание)
  2. Затем создайте Категории для каждого типа
  3. После этого создайте Подкатегории для категорий
  4. В последнюю очередь создайте Статусы
Важные правила:
  • Категории привязаны к типам операций
  • Подкатегории привязаны к категориям
  • Нельзя удалить элемент, если он используется в записях
  • Названия должны быть уникальными в рамках своего типа
